# OpenClaw React Native App

Expo-managed React Native app for OpenClaw.

Current focus:
- Android development now

## Commands

From `apps/react-native`:

```bash
bun install
bun run start
bun run android
```

Optional:

```bash
bun run ios
bun run web
bun run lint
bun run typecheck
```

From repo root:

```bash
pnpm react-native:start
pnpm react-native:android
```
